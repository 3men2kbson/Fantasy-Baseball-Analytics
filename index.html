💡 Cómo encontrar player keys

Los player keys tienen el formato 422.p.XXXXXX donde 422 es el game_id de MLB en Yahoo.

Puedes encontrarlos en la respuesta del endpoint /api/team/.../roster o en la URL de Yahoo Fantasy cuando visitas la página de un jugador.

GET /api/team/{team_key}/roster
# → incluye player_key por jugador

GET /api/league/{key}/free-agents
# → player_key de agentes libres
Configuración
Setup del servidor Python y credenciales Yahoo
🖥️ Iniciar el backend Python
# 1. Entra a la carpeta del backend
$ cd fantasy-baseball/backend

# 2. Instala dependencias
$ pip install -r requirements.txt

# 3. Copia el archivo de variables
$ cp .env.example .env

# 4. Edita .env con tu Client ID y Secret
$ nano .env

# 5. Corre el servidor
$ uvicorn main:app --reload --port 8000
🔑 Pasos Yahoo Developer Console
1
Abre developer.yahoo.com/apps/create
2
Application Name: Fantasy Baseball Analytics
3
Application Type: Web Application
4
Homepage URL: http://localhost:5173
5
En API Permissions → Fantasy Sports → marca Read
6
Callback URL: http://localhost:8000/auth/callback
7
Haz click en Create App y copia el Client ID y Client Secret
8
Pégalos en backend/.env y reinicia el servidor
